# foggy_map

Локальное desktop-приложение для отметки посещенных мест на карте мира.

## Планируемый Стек

- Tauri
- React + TypeScript + Vite
- MapLibre GL JS
- SQLite
- H3 для хранения закрашенных территорий

## Окружение

Зависимости проекта должны устанавливаться локально:

- Node-зависимости - только в `node_modules/` через `npm install`.
- Tauri CLI - как devDependency проекта, запуск через `npm run ...`, без `npm install -g`.
- Python-утилиты, если понадобятся, устанавливаются в локальное `.venv/`.
- Rust toolchain устанавливается на уровне пользователя через `rustup`, но crates и build output проекта не коммитятся.
